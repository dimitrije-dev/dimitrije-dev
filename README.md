## About

I am a final-year Software Engineering student focused on building modern, clean, and practical applications across mobile, web, backend, and AI-assisted systems.

My main focus is mobile development, especially iOS applications with Swift, SwiftUI, MVVM, and local persistence. Alongside mobile development, I also work with full-stack web technologies, Java desktop/web applications, backend APIs, databases, and AI transcription workflows.

I enjoy building products with clear architecture, polished interfaces, and practical user-focused features.
